Callback functions

First thing we will cover is callback functions because these will be essential to know for some of the other topics.

A callback function is a function that is passed as an argument to another function and is executed after some specific code. In other words, instead of passing data as an argument to a function, you're passing a function that can perform some behavior or operation on that data.

Here is a small example:

function doSomething(callback) {

  console.log("something...")

  callback()

}

function helloWorld() {

  console.log("Hello world!")

}

doSomething(sayHello)

In this example we define a function called doSomething() that takes a single argument, which is expected to be a callback function. Inside the doSomething() function we perform some task and then call the callback function.

We call the doSomething() function and pass the sayHello() function as the callback. When doSomething() is executed, it logs the message "Doing something..." to the console and then calls the sayHello() function.


Another example of a callback function is in the setInterval() or/and setTimeout(). This could look something like this:

setTimeout( () => {

  console.log("wait 1 second to excute")

},1000)


Declaring variables

In javaScript const, let and var are used to declare variables.

The const keyword is used to declare a constant variable, which cannot be reassigned or modified. When you declare a variable using const you have to initialize it with a value, and you cannot change that value later in the code. However you are allowed to modify content inside a const array or objects

On the other hand we have the let keyword. This is used to declare a variable that can be reassigned later in the code. You can also also leave a let uninitialized

Lastly we have the var keyword. This keyword is used to declare a variable that can be reassigned just like the let keyword. It has function scope, which means it is accessible throughout the function in which it is defined. Mostly you should avoid using the var keyword unless you have a specific reason to use it.

Destructuring

Destructuring is a feature in JavaScript that allows you to extract values from arrays or objects and assign them to variables. You can extract values from an array by using square brackets [] or from an object by using curly braces {}.

Here are 2 examples:

const arr = [1, 2, 3]

const [a, b, c] = arr


const obj = { name: 'Daniel', age: 22 }

const { name, age } = obj

Arrays in javascript has functions that can manage our data stored in arrays. Here i will give example of the commonly used:

  • map() When you want to transform elements in an array.

  • filter() When you want to select a some elements from an array.

  • find() When you want to select a single element from an array.

Here are a few examples

const arr = [1, 5, 8, 13, 18]


// the find method will return the first number above 10 in the array

// the filter method will return a list of all the number above 10

const foundNumbers = arr.filter(number => number > 10)


// the map method will return an array with all elements timed 2. In this case: [2,10,16,26,36]

Fetch and promises

The fetch() method is used to send HTTP requests to a server and receive responses from it. It takes 2 arguments:

  • First argument is the url. A string representing the URL to which the request is sent.

  • The second argument is an object containing request method, headers, body, etc. The method attribute is by default set to a GET.

The fetch() method returns a promise which an object that represents a value that may not be available yet. Promises are widely used to handle asynchronous operations such as fetches. When an asynchronous operation is started, a promise is returned immediately. This Promise is initially in the pending state, and will transition to be fulfilled or rejected when the operation completes. To handle these promises when can either use .then() and catch() or we can use await/async

Here are some examples of fetches and promises:

fetch("https://pokeapi.co/api/v2/pokemon/1")
.then(res => res.json())
.then(data => console.log(data))
.catch(e => console.log(e))


try{
  const res = await fetch("https://pokeapi.co/api/v2/pokemon/1")
  const data = await res.json()
  console.log(data)
}catch(e){
  console.log(e)
}
