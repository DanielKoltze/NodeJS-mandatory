Terminal

Today i will teach you a few git and npm commands in the terminal.



Git is a powerful version control system that is widely used to collaborate on projects. To use git in the terminal we first have to install it. After the installation we can check if git was succesfully downloaded by writing git --version in the terminal.

Now we are able to use a bunch of git commands! Here is an explanation of the most commonly used once:

1. git init: initializes a new Git repository in the currenty working directory.

2. git add "file/directory": adds changes to the staging area. You can specify a single file or directory, or use ".", "*" or "-A" to add all changes.

3. git commit -m "commit message": commits the changes in the staging area with a commit message.

4. git status: shows the current status of the repository, including which files have been modified or staged.

5. git pull: downloads changes from a remote repository.

6. git push: uploads local changes to a remote repository.


Just like before, we have to download NodeJS before we can use it and the commands that go with it. If this is done succesfully we should be able to check the version by typing node --verision in the terminal.

Once this is installed navigate to your project directory in the terminal and run npm init. This will create a package.json file, which will contain information about your project and its dependencies.

To install packages we can write npm i "package name".
